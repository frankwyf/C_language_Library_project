# Security Policy

## Supported Versions

This project is a learning/portfolio project. Security fixes are best-effort on the latest default branch.

## Reporting a Vulnerability

Please open a private security report through the hosting platform if available, or create an issue without disclosing exploit details publicly.

When reporting, include:

- Affected file/function
- Reproduction steps
- Expected and actual behavior
- Risk impact

I will acknowledge reports as soon as possible and provide mitigation guidance.
